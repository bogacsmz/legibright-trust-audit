# Legibright — brand assets

The product brand for this DataHub trust-audit agent. The **"L" is a ✓ (checkmark)** — the
audit / trust mark — which is exactly the theme: verify before you trust.

## Palette (indigo)
| Token | Hex | Use |
|---|---|---|
| Ink | `#1E1B4B` | primary text, the vertical stroke of the L |
| Indigo | `#4F46E5` | the checkmark stroke, primary accent, links/CTAs |
| Indigo-light | `#6366F1` | secondary accent / highlights |
| Paper | `#FFFFFF` | background |

Use this palette for the demo **title card** and any generated visuals (verdict cards,
diagrams). Verdict colors stay semantic: 🟢 TRUSTWORTHY / 🟡 INCONCLUSIVE / 🔴 NOT TRUSTWORTHY.

## Assets
| File | Use |
|---|---|
| `legibright-lockup.svg` | full logo + wordmark — README header, docs, slide title |
| `legibright-mark.svg` | icon only — demo **watermark**, small in-app icon, favicon fallback |
| `legibright-favicon.svg` | favicon / repo-site icon / the Legibright agent's avatar in DataHub |

## Naming
- **Legibright** — the brand / product display name.
- **Statistical Trust Layer** — what it is (category); package `statistical-trust-layer`, module
  `trust_layer`, CLI `trust-layer`.
- **datahub-trust-audit** — the installable DataHub skill.
- **Auditor** / **Sentinel** — the two component suites.
